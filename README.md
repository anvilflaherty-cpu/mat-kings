# MAT KINGS

Retro arcade wrestling game — Tecmo Bowl meets the mat.

## Play

Open `index.html` in any browser. No install needed.

## Controls

| Key | Action |
|-----|--------|
| Arrow Up/Down | Navigate menus / Pick move |
| Enter / Space | Select |

## How It Works

- Pick your wrestler — each has unique stats (STR, SPD, TEC, STA)
- Pick a move each turn — your opponent picks simultaneously (like Tecmo Bowl plays)
- Move types counter each other:
  - **Defense** beats **Takedown**
  - **Takedown** beats **Throw**
  - **Throw** beats **Pin**
  - **Pin** beats **Defense**
- Stats and move power influence the outcome
- Drain your opponent's stamina to zero for a PIN
- Win by decision (most points) after 3 periods or by PIN
- Beat all 5 opponents to become MAT KING
